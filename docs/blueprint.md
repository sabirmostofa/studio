# **App Name**: CV Alchemist

## Core Features:

- HTML CV Input: Accepts the user's CV as HTML code, directly pasted into the app.
- CV Parsing & Structuring: Parse and intelligently structure the CV data extracted from the HTML to optimize the styling.
- Style Generation with AI: Based on the extracted CV, our tool will apply AI to generate appropriate, professional, and visually appealing styles, and format them with CSS.
- Slick CV Output: The result is presented as a visually refined, CSS-styled CV.
- Downloadable HTML: Users can download the generated CV as a single HTML file.

## Style Guidelines:

- Primary color: Slate blue (#5A7DA6) for a professional, trustworthy impression.
- Background color: Light gray (#F0F2F5), providing a clean and neutral canvas.
- Accent color: Muted lavender (#A090B5), used sparingly for highlights and calls to action, avoiding clich√©s and overpowering brighter tones.
- Headline font: 'Belleza' sans-serif for headings and titles, giving an elegant and artistic touch. Body Font: 'Alegreya' serif, selected for longer chunks of text to improve readability.
- Use a clean, single-column layout for easy readability. Employ generous white space to avoid a cluttered look.
- Minimalist icons (if used) to represent sections such as Experience, Skills, and Education, sourced from a simple line icon set.
- Subtle transitions when the CV is loaded, such as a fade-in effect. Avoid overly complex animations.
